##  Rasa 简单的中文天气查询聊天机器人

基于[luojie1024](https://github.com/luojie1024)Rasa2.0, Rasa X，支持中文城市天气查询

### 依赖包安装

```python
1. 下载python依赖包
pip install -r requirements.txt
2. 下载中文词向量模型，下载后放入data文件夹下
https://pan.baidu.com/s/1kNENvlHLYWZIddmtWJ7Pdg(密码：p4vx)
```

### 项目运行

```pyth
1. 模型训练
rasa train 
2. 运行action server
rasa run actions
3. 运行rasa x
rasa x
```
